# ejercicio2_kivadataset_JoaquinMorenoFernandez
Ejercicio dataset kiva en google colab

Objetivos del ejercicio (que hemos aprendido):

1º Limpiar el data set para eliminar celdas redundantes, con informacion sin valor, o vacías
2º Filtrado del data set, para extraer de manera mas facil y eficiente la información que nos interesa a nosotros (means, medians, meridians, highest of, lowest of...)
3º Construccion de gráficos; que nos ayudan a poder visualizar y presentar la información deseada clara y eficazmente
4º Introduccion de celdas nuevas relevantes; ayudan a complementar la informacion de nuestro data set con conclusiones y posible informacion nueva interesante
